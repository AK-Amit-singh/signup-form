# Responsive Signup Form

## Project Overview

This project is a responsive Signup Form built using **HTML, CSS, and JavaScript**. It provides a clean user interface with form validation and follows a modern signup page design.

## Features

* Responsive design
* Email validation
* Password validation
* Confirm password validation
* Date of Birth field
* Student/Employee selection
* Terms and Conditions checkbox
* Success and error messages
* User-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
signup-form/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## Validation

* Valid email format required.
* Password must contain at least 8 characters, one uppercase letter, one lowercase letter, and one number.
* Confirm Password must match the Password.
* All required fields must be completed before submitting.
